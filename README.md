# AWS Cloud Portfolio

This repository contains a curated set of production-style AWS cloud projects that demonstrate core cloud operations, infrastructure, automation, monitoring, networking, and application architecture skills.

## Portfolio Overview

Each project is organized as a self-contained learning module with objectives, architecture notes, implementation steps, and outcomes. The projects progress from foundational services to more advanced cloud-native and production-ready patterns.

## Project Index
- Project-00-AWS-Account Setup & Security/
- Project 01 - S3 Static Portfolio
- Project 02 - EC2 Web Application
- Project 03 - CloudWatch Monitoring
- Project 04 - CloudTrail Log Analysis
- Project 05 - Lambda Step Functions
- Project 06 - RDS
- Project 07 - Terraform
- Project 08 - Serverless Application
- Project 09 - VPC Network Design
- Project 10 - Production Environment
- Project 11 - Capstone Banking Application

## Skills Covered

- AWS compute, storage, networking, and database services
- IAM, security, and least-privilege access
- Monitoring, logging, and observability
- Infrastructure as Code with Terraform
- Serverless application design
- Production readiness and operational excellence
