# Project 09 - VPC Network Design

## Objective
Design and implement a secure virtual network topology with subnets, route tables, NAT, and security boundaries.

## Key Concepts
- VPC, subnets, route tables, and internet gateway
- Public versus private subnets
- Network access control and routing

## Suggested Deliverables
- VPC architecture diagram or description
- Subnets and routing configuration
- Security and connectivity considerations
