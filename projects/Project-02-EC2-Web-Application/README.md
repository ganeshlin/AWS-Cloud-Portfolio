# Project 02 - EC2 Web Application

## Objective
Deploy a web application on Amazon EC2 with a Linux-based web server, security group rules, and basic monitoring.

## Key Concepts
- EC2 instance provisioning
- Security groups and SSH access
- Web server installation and service management

## Suggested Deliverables
- EC2 instance running a sample web app
- Security group configured for HTTP/HTTPS and SSH
- Deployment notes and screenshots

## Sample Application
- [app.py](app.py) provides a simple Python web app example
- [requirements.txt](requirements.txt) lists the Python dependencies
