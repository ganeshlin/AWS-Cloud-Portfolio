# Project 00 - AWS Account Setup & Security

## Objective
Set up a secure AWS account foundation with IAM best practices, billing awareness, and basic security controls.

## Key Concepts
- AWS account root and billing setup
- IAM users, groups, and roles
- MFA and least-privilege access
- Budgets and alerts

## Suggested Deliverables
- Secure IAM setup
- MFA enabled for privileged access
- Budget alert configuration
