# Project 01 - S3 Static Portfolio

## Objective
Host a static website using Amazon S3 and enable public access through a custom domain or S3 website endpoint.

## Key Concepts
- Amazon S3 static website hosting
- Bucket policies and public access configuration
- CloudFront optional enhancement

## Suggested Deliverables
- S3 bucket configured for static website hosting
- HTML/CSS/JS assets uploaded
- README with deployment steps and architecture notes

- [source-code](source-code) contains the website files

- [architecture](architecture) contains architecture notes and diagrams
