# Project 07 - Terraform

## Objective
Provision AWS infrastructure using Terraform with reusable modules and version-controlled configuration.

## Key Concepts
- Terraform configuration and state
- Providers and resources
- Infrastructure as Code best practices

## Suggested Deliverables
- Terraform configuration for one or more AWS services
- Variables, outputs, and modules
- Deployment and destroy workflow documented

## Starter Templates
- [main.tf](main.tf) defines a basic S3 bucket and website configuration
- [variables.tf](variables.tf) contains configurable inputs
- [outputs.tf](outputs.tf) exposes useful outputs
